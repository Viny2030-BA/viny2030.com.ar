```
### **4. Configuración alternativa en Render**

Si el archivo definitivamente está en `python/requirements.txt`, cambia en Render:
```
Root Directory: (vacío)
Build Command:  pip install -r python/requirements.txt
```

O usa:
```
Root Directory: python
Build Command:  pip install -r requirements.txt
Start Command:  cd .. && gunicorn app:app
